# TradeReview - 股票关注池

密切观察列表中的股票，并提醒是否可以购买

## 功能特性

1. **清爽的UI设计** - 使用现代化的Pencil风格设计，界面简洁美观
2. **价格监控** - 实时监控关注池中的股票价格，自动提醒价格变化
3. **智能预警** - 根据价格接近程度显示不同颜色：
   - 🔴 红色(立即提醒): 当前价格达到或低于卖出价格
   - 🟠 橙色(接近目标): 当前价格在卖出价格5%以内
   - 🟡 黄色(注意观察): 当前价格在卖出价格10%以内
   - 🟢 绿色(正常): 当前价格高于卖出价格10%以上
4. **实时数据获取** - 从免费股票数据接口自动获取股票名称和当前价格
   - 主要数据源：新浪财经 API
   - 备用数据源：腾讯财经 API
   - 支持所有A股股票（沪深两市）
5. **完整信息展示** - 显示股票代码、名称、当前价格和卖出价格
6. **自动价格更新** - 每5秒自动从API获取最新价格
7. **数据持久化** - 使用LocalStorage保存数据，刷新页面不会丢失

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 填写股票信息（仅需两个字段）：
   - **股票代码**（6位数字，如：000001、600519、000066）- 输入后自动获取股票信息
   - **卖出价格**（目标价格）- 您希望在此价格卖出
3. 系统会自动从股票API获取股票名称和当前价格并显示预览
4. 点击"添加到关注池"按钮
5. 查看关注池中的股票，系统会根据价格自动显示不同的预警颜色
6. 系统每5秒自动更新所有股票的最新价格
7. 也可以手动点击"更新当前价格"按钮立即刷新单个股票价格

### 支持的股票

- 支持所有沪市股票（代码以6开头，如：600519）
- 支持所有深市股票（代码以0或3开头，如：000001、000066、300750）
- 数据来源：新浪财经、腾讯财经等免费API接口

## 技术实现

- 纯HTML、CSS、JavaScript实现，无需后端服务器
- 响应式设计，支持移动端和桌面端
- 使用LocalStorage进行数据持久化
- 集成新浪财经和腾讯财经API获取实时股票数据
- 自动重试机制：如果主API失败，自动切换到备用API
- 每5秒自动刷新所有股票价格
